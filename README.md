# 🏥 Ulcer Health Risk Analysis Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze ulcer risk, patient health metrics, and clinical patterns to generate actionable healthcare insights.

---

## 📊 Project Overview

The **Ulcer Health Risk Analysis Dashboard** helps in understanding patient conditions by visualizing key health indicators such as BMI, hemoglobin levels, ulcer severity, and pain patterns.

This project focuses on transforming raw healthcare data into meaningful insights that support better decision-making and risk assessment.

---

## 🎯 Objective

* Analyze ulcer-related health data across multiple dimensions
* Identify high-risk patient groups
* Monitor key health indicators
* Support data-driven clinical decisions

---

## ⚡ Key Features

### 📌 KPI Overview

* Total Patients
* Average BMI
* Average Hemoglobin Level

### 📌 Ulcer Analysis

* Distribution of ulcer depth levels (Erosion, Superficial, Deep)
* Helps identify severity patterns

### 📌 Patient Insights

* Age group distribution
* Ulcer history (Previous, Recurrent, None)

### 📌 Medication Analysis

* Patient distribution across treatments like NSAIDs and Aspirin

### 📌 Pain Pattern Analysis

* Postprandial
* Fasting
* Nocturnal
* Mixed
